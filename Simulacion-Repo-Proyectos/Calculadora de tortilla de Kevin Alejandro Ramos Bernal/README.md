# Spanish Omelette Calculator

Simple web application developed with **HTML, CSS and JavaScript** that automatically calculates the approximate amount of ingredients needed to cook a **Spanish omelette (with or without onion)** based on the number of people.

---

## Project Goal

The goal of this project is to provide a practical and intuitive tool to adjust ingredient quantities for a Spanish omelette depending on the number of diners, helping to avoid waste and making cooking planning easier.

This project is focused on learning **basic JavaScript**, DOM manipulation and lightweight web app structure.

---

## How the Application Works

1. The user enters the number of people.
2. Selects whether the omelette is **with onion or without onion**.
3. The application automatically calculates:

   * Amount of potatoes
   * Number of eggs
   * Amount of onion (if selected)
4. Results are displayed instantly on screen.

All calculations are performed in the browser using JavaScript, with no server or database required.

---

## How to Use the Page

1. Open the `index.html` file in any modern browser.
2. Enter the number of diners.
3. Select the omelette type.
4. Check the recommended ingredient quantities displayed.

No installation or internet connection required.
