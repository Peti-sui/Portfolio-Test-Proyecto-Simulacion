# Entrada y Procesamiento de Números

Aplicación web sencilla desarrollada con **HTML, CSS y JavaScript** que permite al usuario ingresar un **número no negativo** y procesarlo para mostrar resultados dinámicos en pantalla.

---

## Objetivo del Proyecto

El objetivo de esta página es practicar el uso de formularios en HTML y la validación de datos, así como la **manipulación del DOM con JavaScript**, mostrando resultados basados en un número ingresado por el usuario.

Este proyecto forma parte de una actividad académica enfocada al aprendizaje de **JavaScript básico** y la interacción con el usuario.

---

## Funcionamiento de la Aplicación

1. El usuario ingresa un **número que no sea negativo** en el campo de entrada.
2. Al presionar el botón **Enviar**, el formulario se procesa mediante JavaScript.
3. El script realiza las operaciones o validaciones correspondientes.
4. Los resultados se muestran dinámicamente en los contenedores:

   - `resultado1`
   - `resultado2`

Todo el procesamiento se realiza del lado del cliente, directamente en el navegador.

---

## Cómo Utilizar la Página

1. Abrir el archivo `index.html` en cualquier navegador moderno.
2. Ingresar un número no negativo en el campo de texto.
3. Presionar el botón **Enviar**.
4. Visualizar los resultados generados en pantalla.

No requiere instalación ni conexión a internet.

---

## Tecnologías Utilizadas

- HTML5  
- CSS3  
- JavaScript (Vanilla)
