# Separador de Decenas y Unidades

Aplicación web sencilla desarrollada con **HTML, CSS y JavaScript** que permite introducir un número de **máximo dos cifras** y muestra por separado sus **decenas y unidades**.

---

## Objetivo del Proyecto

El objetivo de esta página es practicar el uso de **formularios HTML**, la **manipulación del DOM** y el procesamiento de datos numéricos con **JavaScript**, separando correctamente las decenas y unidades de un número introducido por el usuario.

Este proyecto forma parte de la **Actividad 5** y está enfocado al aprendizaje de conceptos básicos del desarrollo web.

---

## Funcionamiento de la Aplicación

1. El usuario introduce un número de **dos cifras** (entre 10 y 99).
2. El número es procesado mediante JavaScript.
3. La aplicación separa:
   - La cifra de las **decenas**
   - La cifra de las **unidades**
4. Los resultados se muestran dinámicamente en pantalla en dos párrafos distintos.

Todo el proceso se realiza en el navegador, sin necesidad de servidor ni base de datos.

---

## Cómo Utilizar la Página

1. Abrir el archivo `index.html` en cualquier navegador moderno.
2. Introducir un número de dos cifras en el campo correspondiente.
3. Observar cómo se muestran las decenas y las unidades en pantalla.

No requiere instalación ni conexión a internet.

---

## Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
